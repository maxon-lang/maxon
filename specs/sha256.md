# SHA-256

## Documentation

Compute the SHA-256 cryptographic hash of a byte array. Returns a 32-byte `ByteArray` containing the hash digest.

**Signature:** `sha256(data ByteArray) returns ByteArray`

**Parameters:**
- `data` - The input bytes to hash

**Returns:** A 32-byte `ByteArray` containing the SHA-256 digest

**Example:**

```maxon
function main() returns ExitCode
	let data = ByteArray.create()
	data.push(0x61)
	data.push(0x62)
	data.push(0x63)
	let hash = sha256(data)
	// hash is the SHA-256 of "abc"
	// first byte is 0xba = 186
	let first = try hash.get(0) otherwise 0
	print("{first}\n")
	return 0
end 'main'
```

**Notes:**
- Implements the FIPS 180-4 SHA-256 algorithm
- Always returns exactly 32 bytes
- The empty input hashes to `e3b0c442...7852b855`

function main() returns ExitCode
	let data = ByteArray.create()

	data.push(0x62)
	data.push(0x63)

### Padding boundary: 55 zero bytes (max single-block message)

<!-- test: sha256.55-zero-bytes -->
```maxon
function main() returns ExitCode
	let data = ByteArray.create()
	var i = 0
	while i < 55 'fill'
		data.push(0)
		i = i + 1
	end 'fill'
	let hash = sha256(data)
	// SHA-256(55 * 0x00) = 02779466...
	// first byte = 0x02 = 2, last byte = 0xf7 = 247
	let first = try hash.get(0) otherwise 255
	let last = try hash.get(31) otherwise 0
	print("{first} {last}\n")
	return 0
end 'main'
```
```exitcode
0
```
```stdout
2 247
```
