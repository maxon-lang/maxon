# String Literals

## Documentation

String literals are text enclosed in double quotes.

### Syntax

```maxon
var message = "Hello, World!"
```
### Escape Sequences

- `\n` - Newline
- `\t` - Tab
- `\\` - Backslash
- `\"` - Double quote

### Example

```maxon
  var msg = "Hello!\n"
```
