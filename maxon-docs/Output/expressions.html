Expressions

Expressions combine values, operators, and function calls to compute results.

Simple Expressions

2 + 3
x * y
a > b

Compound Expressions

Combine multiple operations:

(2 + 3) * 5
a + b * c - d
x > 0 and y < 10  // When logical operators added

Expressions with Function Calls

5 + add(3, 4)
sqrt(x * x + y * y)

Type Compatibility

Operands must have compatible types:

  • int + int → int
  • float + float → float
  • int + float → float (int promoted)
  • int > int → bool

Literals

Literals are constant values used directly in code.

Integer Literals

42
-17
0

Float Literals

Must include decimal point:

3.14
-2.5
0.0

Character Literals

Single character in single quotes:

'A'
'z'
'\n'

String Literals

Text in double quotes:

"Hello, World!"
"Line1\nLine2"

Boolean Literals

true
false

Parentheses

Parentheses group expressions and control evaluation order.

Syntax

(expression)

Example

function main() int
    var a = 2 + 3 * 4      // 14 (multiply first)
    var b = (2 + 3) * 4    // 20 (add first)
    print(a)
    return b
end 'main'

ExitCode: 20
Stdout: 14
