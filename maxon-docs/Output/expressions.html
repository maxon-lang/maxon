Array Element Method Call

Method Calls on Array Elements

You can call methods directly on array elements using the syntax arr[i].method(args).

Array Struct Element Member Access

Type Field Access on Array Elements

You can access type fields directly on array elements using arr[i].field syntax.

Expressions

Expressions combine values, operators, and function calls to compute results.

Simple Expressions

2 + 3
x * y
a > b

Compound Expressions

Combine multiple operations:

(2 + 3) * 5
a + b * c - d
x > 0 and y < 10  // When logical operators added

Expressions with Function Calls

5 + add(3, 4)
sqrt(x * x + y * y)

Type Compatibility

Operands must have compatible types:

  • int + int → int
  • float + float → float
  • int + float → float (int promoted)
  • int > int → bool

Literals

Literals are constant values used directly in code.

Integer Literals

Decimal integers:

42
-17
0

Hexadecimal integers (prefix 0x):

0xff
0x1a2b
0x0

Binary integers (prefix 0b):

0b1010
0b11111111
0b0

Octal integers (prefix 0o):

0o777
0o52
0o0

Underscore separators can be used for readability in any integer literal:

1_000_000
0xff_ff
0b1111_0000
0o77_77

Float Literals

Must include decimal point:

3.14
-2.5
0.0

Character Literals

Single character in single quotes:

'A'
'z'
'\n'

String Literals

Text in double quotes:

"Hello, World!"
"Line1\nLine2"

Boolean Literals

true
false

Method Call On Parameter

Method Calls on Parameters of the Same Type

When writing a method, you can call other methods on parameters that have the same type as self.

Parentheses

Parentheses group expressions and control evaluation order.

Syntax

(expression)

Example

function main() returns int
    var a = 2 + 3 * 4      // 14 (multiply first)
    var b = (2 + 3) * 4    // 20 (add first)
    print("{a}")
    return b
end 'main'
Expected Output
14
Exit Code: 20
