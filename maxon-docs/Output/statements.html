Assignment

The assignment operator = updates the value of a mutable variable.

Syntax

variable = expression

Example

function main() int
    var x = 10
    x = 20          // Update x
    x = x + 5       // x is now 25
    return x
end 'main'
Expected Output
Exit Code: 25

Restrictions

  • Cannot assign to let variables
  • Variable must be declared with var
  • Expression type must match variable type

Return Statement

The return statement exits the current function and optionally returns a value to the caller.

Syntax

return expression

Example

function isPositive(x int) bool
    if x > 0 'check'
        return true
    end 'check'
    return false
end 'isPositive'
