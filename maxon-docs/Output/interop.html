Extern Functions

The extern keyword declares functions defined outside your Maxon code, such as system APIs or C library functions.

Syntax

extern function name(param1 type1, param2 type2) returnType

Note: No function body or end statement.

Example

extern function TestFunc(x int, p ptr, c char) returns int

function main() returns int
    return 0
end 'main'
Expected Output
Exit Code: 0
