Exp

The exp() function calculates e raised to a power (e^x).

Syntax

exp(x)

Parameters:

  • x - The exponent (float)

Returns e raised to the power of x, where e ≈ 2.71828 (Euler's number).

Example

function main() int
    var result = exp(0.0)  // e^0 = 1
    return trunc(result)
end 'main'

Notes

  • Returns 1.0 for x = 0
  • Can handle both positive and negative exponents
  • Result is always positive
  • Integer inputs are automatically promoted to float

Format Float

The format_float_array() function converts a float to a string with specified decimal precision.

Syntax

format_float_array(value, buffer, precision)

Parameters:

  • value - The float to convert
  • buffer - Character array to store result
  • precision - Number of decimal places

Returns the number of characters written to the buffer.

Example

function main() int
    var buffer = [50]char
    var len = format_float_array(3.14159, buffer, 2)
    // buffer now contains "3.14"
    return len  // Returns 4
end 'main'

Format Int

The format_int_array() function converts an integer to a string representation.

Syntax

format_int_array(value, buffer)

Parameters:

  • value - The integer to convert
  • buffer - Character array to store result (must be at least 12 bytes)

Returns the number of characters written to the buffer.

Example

function main() int
    var buffer = [12]char
    var len = format_int_array(42, buffer)
    // buffer now contains "42"
    return len  // Returns 2
end 'main'

Notes

  • Buffer must be at least 12 bytes to handle maximum negative value (-2147483648)
  • No null terminator is added
  • Returned length can be used to determine actual string size

Log

The log() function calculates the natural logarithm (ln) of a number.

Syntax

log(x)

Parameters:

  • x - The value to take the logarithm of (must be positive)

Returns the natural logarithm of x.

Example

function main() int
    var e = 2.71828
    var result = log(e)  // ln(e) ≈ 1.0
    return trunc(result)
end 'main'

Notes

  • Input must be positive (x > 0)
  • Returns 0.0 for x = 1.0
  • Returns 0.0 for invalid inputs (x <= 0)
  • Uses natural logarithm (base e), not base 10

Pow

The pow() function raises a number to a power.

Syntax

pow(base, exponent)

Parameters:

  • base - The base number (float)
  • exponent - The exponent (float)

Returns base raised to the power of exponent.

Example

function main() int
    var result = pow(2.0, 3.0)  // 2^3 = 8
    return trunc(result)
end 'main'

Notes

  • Both parameters are floats
  • Integer inputs are automatically promoted to float
  • Returns float result
  • Special cases:
    • pow(x, 0.0) returns 1.0 for any x
    • pow(0.0, y) returns 0.0 for positive y
    • pow(1.0, y) returns 1.0 for any y

Print Float

The print_float() function outputs float values with specified decimal precision.

Syntax

print_float(value, precision)

Parameters:

  • value - The float to print
  • precision - Number of decimal places

Example

function main() int
    var pi = 3.14159
    print_float(pi, 2)     // Prints: 3.14
    print_float(pi, 5)     // Prints: 3.14159
    
    var x = -2.5
    print_float(x, 1)      // Prints: -2.5
    
    return 0
end 'main'

Print Function

The print() function outputs integer values to standard output (console).

Syntax

print(value)

Where value is an int expression.

Example

function main() int
    var x = 42
    print(x)        // Prints: 42
    print(10 + 5)   // Prints: 15
    print(100)      // Prints: 100
    return 0
end 'main'

Each call to print() outputs the value followed by a newline.

Stdlib Autodiscovery

The standard library is automatically discovered and linked when you use its functions.

How It Works

When you call a stdlib function like pow(), sqrt(), or print(), the compiler:

  1. Searches the stdlib/ directory
  2. Finds the function definition
  3. Compiles it automatically
  4. Links it into your program

No imports or includes needed!

Example

function main() int
    // pow() is automatically found in stdlib/math/
    var result = pow(2.0, 3.0)
    return result as int
end 'main'

Transitive Dependencies

If a stdlib function depends on other stdlib functions, they're also discovered automatically. For example, pow() uses log() and exp(), which are all linked automatically.

Stdlib Namespaces

The standard library organizes functions into logical namespaces.

Available Namespaces

  • sys - System operations (I/O, process control)
  • fs - File system and stream operations
  • fmt - String formatting and conversion
  • math - Mathematical functions

Example

namespace fs 'fs'
    function stdout() ptr
        // Returns stdout handle
    end 'stdout'
    
    function write(handle ptr, buffer ptr, length int) int
        // Write bytes to file handle
    end 'write'
end 'fs'

function main() int
    let out = stdout()
    var msg = "Hello\n"
    write(out, msg, 6)
    return 0
end 'main'
