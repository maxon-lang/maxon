Comparison operators compare two values and return true or false.

Operators

  • = - Equal to
  • != - Not equal to
  • < - Less than
  • > - Greater than
  • <= - Less than or equal to
  • >= - Greater than or equal to

Example

function main() int
    var x = 10
    var y = 20
    
    if x < y 'test1'
        if x != y 'test2'
            return 1
        end 'test2'
    end 'test1'
    
    return 0
end 'main'
