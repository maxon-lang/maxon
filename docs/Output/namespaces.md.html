Namespaces group related functions together and help organize code.

Syntax

namespace name 'name'
    // function declarations
end 'name'

Example

namespace math 'math'
    function add(a int, b int) int
        return a + b
    end 'add'
    
    function multiply(x int, y int) int
        return x * y
    end 'multiply'
end 'math'

function main() int
    return add(3, 4)  // Called from within namespace
end 'main'
