The char type stores a single character as an 8-bit value.

Syntax

var letter = 'A'
let newline char = '\n'

Character literals are enclosed in single quotes.

Example

function isUppercase(c char) bool
    var code = c as int
    if code >= 65 'check'
        if code <= 90 'check2'
            return true
        end 'check2'
    end 'check'
    return false
end 'isUppercase'

function main() int
    if isUppercase('Z') 'test'
        return 1
    end 'test'
    return 0
end 'main'
