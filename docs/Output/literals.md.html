Literals are constant values used directly in code.

Integer Literals

42
-17
0

Float Literals

Must include decimal point:

3.14
-2.5
0.0

Character Literals

Single character in single quotes:

'A'
'z'
'\n'

String Literals

Text in double quotes:

"Hello, World!"
"Line1\nLine2"

Boolean Literals

true
false
